## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ShettyPratik13/linkedin_post_formatter.git
cd linkedin_post_formatter
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

1. **Compose Your Post**: Type or paste your content in the text area on the left
2. **Add Emojis**: Click on any emoji button to add it to your post
3. **Preview**: See how your post looks in the preview pane on the right
4. **Copy**: Click the "Copy to Clipboard" button to copy your formatted post
5. **Paste**: Go to LinkedIn and paste your beautifully formatted post!

## Building for Production

To create a production build:

```bash
npm run build
```

The optimized files will be in the `dist` directory.

## Tips for Great LinkedIn Posts

- Use 3-5 emojis to increase engagement
- Break text into short paragraphs for better readability
- Add relevant hashtags at the end
- Keep your message clear and concise
- Use the preview to check formatting before posting
